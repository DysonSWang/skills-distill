## 概述

许诺（又称"班长"）是情感导师，其表达风格兼具专业性与草根性，擅长用通俗语言讲解复杂的两性关系理论。整体语气介于朋友建议与专家指导之间，既不学究气也不过度娱乐化。

---

## 一、高频用词与句式

### 1. 引导语与开场白

| 句式 | 频率 | 示例 |
|------|------|------|
| "这节课程/节课" | 极高 | "这节课程给大家讲讲如何跟女生讲故事" |
| "那/那么" | 极高 | "那么既然你下定决心来到了这里" |
| "首先/第一/第二/第三" | 极高 | "第一个叫障碍区"、"第二叫做拐点" |
| "给大家/你们" | 高 | "给大家分享两个最重要的东西" |
| "比如说/比如" | 高 | "比如说你去尝试主动的社交" |
| "举个最简单的例子" | 中 | "举个最简单的例子就是..." |

### 2. 观点引出语

| 句式 | 功能 | 示例 |
|------|------|------|
| "你会发现" | 引出观察结论 | "你会发现它分为两种" |
| "所以" | 引出因果结论 | "所以量变引起质变，是一定的" |
| "其实" | 引出真实/重点内容 | "其实心态是最基本的东西" |
| "简单一点说/简单来说" | 简化复杂概念 | "简单一点说，你就把我每一个流程该犯的错误都犯一遍就好了" |
| "也就是说" | 解释说明 | "也就是说，你正在经历障碍区" |
| "这个就是" | 定义概念 | "这个就是障碍区" |
| "那么" | 开启新话题 | "那么在正式开始学习之前" |

### 3. 强调与总结语

| 句式 | 用法 | 示例 |
|------|------|------|
| "一定/绝对" | 确定性强调 | "吸引是让女生愿意跟你深入接触的原始动力" |
| "这就是为什么..." | 解释原因 | "这就是为什么我们很多兄弟在和女生聊天..." |
| "所以呢" | 引出结论 | "所以呢，大家不要盲目地去放大吸引的作用" |
| "记住" | 重点提醒 | "记住，仅仅只是一个原始动力而已" |

### 4. 过渡与衔接

| 句式 | 功能 | 示例 |
|------|------|------|
| "那么" | 话题转换 | "那么第一个方面..." |
| "那这个" | 指代前文 | "那这个故事呢，就很好的表达了..." |
| "好/好了" | 结束当前话题 | "好了，这节课程给大家讲到这里" |
| "我们下节课见" | 结束语 | 几乎每个视频结尾 |

## 四、确定性表达风格

### 1. 斩钉截铁型

| 表达 | 原文语境 |
|------|----------|
| "追女生不靠追，要靠吸引" | 核心方法论 |
| "恋爱是一种技能，是可以学得会的" | 核心理念 |
| "量变引起质变，是一定的" | 学习规律 |

### 2. 保留余地型

| 表达 | 原文语境 |
|------|----------|
| "可能"、"也许" | 描述可能性 |
| "一般的"、"基本上" | 限制适用范围 |

## 五、场景化表达差异

| 场景 | 风格特点 |
|------|----------|
| 心态建设 | 励志、温和、强调过程；善用"障碍区→拐点→状态回落"三段式模型 |
| 技巧教学 | 直接、步骤化；善用"第一、第二、第三"结构 |
| 案例分析 | 生动、比喻丰富；从进化心理学角度解释 |
| 长期关系 | 严肃、宏观视角；强调"掌控感"和"投资" |

## 八、核心DNA总结

**标志性表达习惯：**
1. 几乎每个视频以"这节课程/节课"开场
2. 善用"第一、第二、第三"结构化表达
3. 结尾必有"总结"和"我们下节课见"
4. 大量使用"那么"、"所以"作为过渡

**核心金句：**
- "追女生不靠追，要靠吸引"
- "恋爱是一种技能"
- "量变引起质变，是一定的"
