## 核心公式

```
正确阶段 × 正确行动 = 成功恋爱
```

**公式解读**：
- **阶段意识**：聊天、约会、表白每个环节都有最佳时机
- **阶段错位**：跳步=失败，约会前非要聊天聊很久，约会后非要确认关系
- **核心思维**：知道自己在哪个阶段，该做什么，不该做什么

## 吸引三层次

凡哥把吸引分为三个层次，从外到内，层层深入。

| 层次 | 内容 | 说明 |
|------|------|------|
| **第一层** | 个人形象 | 颜值、穿搭、干净整洁 |
| **第二层** | 物质和生活 | 工作、家庭、社交圈 |
| **第三层** | 内在价值 | 情商、性格、三观 |

**核心观点**：
- 单纯靠外在形象无法形成长期稳定吸引
- 三层都要建设，但有顺序——先外后内
- 最低要求：可以不帅，但不能脏乱差

## 聊天六步法

**核心定位**：聊天是工具，不是目的。聊天的唯一目的就是邀约。

```
第一步：开场白（表情/问题）
第二步：信息交换（职业/爱好/家乡）
第三步：价值展示（生活方式/社交认证）
第四步：情绪调动（调侃/冷读/推拉）
第五步：模糊邀约（"改天一起XX"）
第六步：敲定时间（具体时间地点）
```

**核心原则**：
- 聊天7-10天还没约出来=聊死了
- 聊得越久，好奇心越低
- 心动的感觉只能见面产生，聊天聊不出心动

## 女性分类（测试型 vs 投资型）

凡哥的女性分类维度与其他导师不同，核心看"长期vs短期"。

| 类型 | 核心特征 | 社交状态 | 注意力 | 相处策略 |
|------|---------|---------|---------|---------|
| **测试型** | 喜欢短期关系 | 社交圈广，爱社交活动 | 不集中，花心 | 不能太认真，快速推进 |
| **投资型** | 喜欢长期关系 | 单纯，宅，生活圈子小 | 集中，专注 | 慢慢来，给足安全感 |

**判断标准**（3条核心）：

| 标准 | 测试型信号 | 投资型信号 |
|------|-----------|-----------|
| **1. 感情史** | 3段以内短期（<3个月），频繁换对象 | 1-2段长期（>1年），稳定 |
| **2. 聊天模式** | 话题跳跃，每3-5句换主题，聊不深 | 话题专注，能就一个话题聊很久 |
| **3. 社交圈** | 朋友圈广，常发自拍/社交/夜生活 | 朋友圈小，少自拍，多工作/静态 |

**实战判断技巧**：
```
问："你平时周末一般干嘛？"
- 测试型答："约朋友逛街/去酒吧/参加活动"（社交活动多）
- 投资型答："在家看书/追剧/偶尔健身"（静态活动多）

问："你谈过几段恋爱？"
- 测试型答："记不太清了，好几段吧"（短期居多）
- 投资型答："两三段吧，每段都挺长的"（长期居多）

看朋友圈自拍频率：
- 测试型：天天发自拍，社交互动多
- 投资型：偶尔发，生活相关，不频繁
```

**核心结论**：
- 2条以上符合测试型 → 快速推进，不要拖
- 2条以上符合投资型 → 慢慢来，给足安全感

## 恋爱思维核心

### 吸引是一切的前提

**定义**：让女生对你感兴趣，愿意跟你建立联系。

**吸引的三个前提**：
1. 她不能反感你（最低要求）
2. 她对你产生好奇心（中配要求）
3. 她主动愿意了解你（最高要求）

### 正确阶段做正确的事

| 阶段 | 核心任务 | 错误做法 |
|------|---------|---------|
| 吸引 | 建立好奇，展示价值 | 跳过直接表白 |
| 聊天 | 邀约服务，不是追她 | 聊太久不约 |
| 约会 | 升级关系，手牵手 | 只吃饭看电影 |
| 确认 | 关系明朗化 | 拖太久不确定 |

---

## 邀约的正确方式

**核心原则**：邀约必须建立在安全感和舒适感基础上。

**邀约失败的原因**：
1. 太过直接（"出来吃个饭吧"）
2. 缺乏铺垫（没有建立安全感就约）
3. 暴露需求感（太渴望，太着急）

**正确邀约流程**：
1. 聊天建立基本了解
2. 交换价值信息（让她了解你）
3. 发朋友圈展示生活（安全感）
4. 模糊邀约试探（"改天一起XX"）
5. 具体邀约敲定

## 约会完整流程

### 约会前准备

| 准备项 | 内容 |
|-------|------|
| 外在形象 | 干净整洁，搭配协调 |
| 约会地点 | 提前考察，准备备选方案 |
| 路线规划 | 预判交通，停车位置 |
| 整体节奏 | 你来带领，不是跟着她走 |

### 约会中

| 阶段 | 时长 | 核心任务 |
|------|------|---------|
| 见面 | 5分钟 | 确认是她，调整状态 |
| 转场 | 10分钟 | 走向餐厅，边走边聊 |
| 用餐 | 30分钟 | 轻松聊天，不查户口 |
| 散步 | 20分钟 | 情绪升级，制造独处 |
| 告别 | 5分钟 | 留余韵，约定下次 |

### 约会铁律

**核心铁律**（3条）：

| 铁律 | 操作 | 金句 |
|------|------|------|
| **坏的选择 > 不敢选择** | 犹豫不决比选错更糟糕，你不做决定就是在浪费两个人的时间 | 「宁可做一个错误的决定，也不要做零决定。」 |
| **你带领她** | 约会的节奏、地点、话题都由你发起，她只需要跟随 | 「不是她选去哪里，而是你带她去哪里。」 |
| **控制节奏** | 不是你追她，是她在跟你走——你永远是那个给方向的人 | 「约会不是面试，是你在主导的一场游戏。」 |

**约会节奏核心**：
```
见面5分钟 → 确认是她，调整状态
转场10分钟 → 边走边聊，营造独处
用餐30分钟 → 轻松聊天，不查户口
散步20分钟 → 情绪升级，制造心动
告别5分钟 → 留余韵，约定下次
```

**约会中的三个不要**：
1. 不要问她"你想去哪里"——这是你的责任
2. 不要一直等她表态——主动是你的义务
3. 不要只吃饭看电影——必须转场，必须升级

## 爱情四象限理论

凡哥用两个维度（快乐+成长）划分四种爱情模式：

| 类型 | 特征 | 评价 |
|------|------|------|
| **享乐型** | 低成长+高快乐 | 好开始，不一定长久 |
| **互惠型** | 高成长+中快乐 | 稳定，适合长期 |
| **消耗捆绑型** | 低快乐+低成长+控制 | 糟糕，警惕这4个信号 |
| **成长治愈型** | 高成长+高快乐 | 理想型，长期目标 |

**消耗捆绑型预警信号**（符合2条就要警惕）：
1. 在一起后越来越不快乐
2. 对方总贬低你，让你越来越没自信
3. 对方限制你的社交圈
4. 对方总提你做不到的要求

## 聊天技巧

### 调戏与赞美

**核心原则**：
- 调戏 = 调情 + 戏弄
- 调戏是快速升级关系的方式
- 单向主动 = 调戏，双向互动 = 调情

**调戏的正确方式**：
```
角色扮演 + 顺势肢体接触
例："小妹妹，一个人害不害怕，要不要哥哥保护你"
     → 顺势轻轻拉一下她的手
```

**调戏的雷区**：
- 不要在对方没准备好时动手动脚
- 不要让对方感到不舒服或被冒犯
- 调戏的目的是升级关系，不是占便宜

### 冷读术

**定义**：在不了解对方的情况下，说出一些普遍性的人格特征，让对方觉得"你很懂我"。

**冷读的正确方式**：
```
"你看起来很开朗，但有时候也会突然安静下来"
"你是个外表活泼但内心敏感的人"
```

**冷读的原理**：大多数人都会对号入座，因为这些话适用于大多数人。

### 情绪调动

**核心**：让她的情绪起伏波动，而不是一直平稳。

| 技巧 | 操作 | 效果 |
|------|------|------|
| 推拉 | 先否定再肯定 | 让她猜不透你 |
| 调侃 | 针对她的特点开玩笑 | 制造互动感 |
| 惊喜 | 突然说一些意外的话 | 情绪冲击 |
