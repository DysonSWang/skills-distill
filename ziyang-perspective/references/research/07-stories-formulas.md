```
→ 应对：通过服从性测试后，该推进就推进，该要约就要约
```

### 2.8 普通朋友升级好朋友的核心（内部课新增）

> 📹 聊什么·第十一节课：普通朋友升级好朋友

```
普通朋友和好朋友的界限：
就是你们有没有聊过1-2个感性的夜晚

你们聊天话题一直停留在浅层的：
吃了没？睡了没？今天吃了什么东西？明天想去哪家店？

只聊这些东西，你们的关系是生不上去的

你们必须经历一到两个感性的夜晚
那天月亮很远，那天夜色很好，你们彼此都很无聊
你们交流了一些对事务的看法、对彼此的感情观、对事业观
对彼此的未来的一些看法
这个时候你们才能变成好朋友
```

### 2.9 暧昧关系0-1推进策略（内部课新增）

> 📹 聊什么·第十七节课：暧昧关系的分值

```
暧昧关系有刻度，最浅0.2，最深1
你不要飞0进1，不要好朋友聊着聊着突然表白
你要刻度丝滑一点：

0.2 = 一句早安晚安（互道早晚安）
0.3~0.4 = 互取昵称（小猪猪、小蓝莓、小狗狗）
0.5 = 土味情话（这辈子太轻压不住想你的心；你的眼睛像繁星，但我的眼睛更好看，因为我的眼里有你）
0.7~0.8 = 开车/聊骚

每个阶段有明确的刻度，不能跳步
```
