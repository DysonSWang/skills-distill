# 投资人评审团 · Investment Committee OS

> 「不让一个人的盲区成为你的死穴。」

---

## 评审团成员分工

| 投资人 | 机构 | 核心特长 | 适用场景 |
|--------|------|---------|---------|
| **徐小平** | 真格基金 | 看人（30分钟激动测试） | 天使轮/早期看创始人 |
| **熊晓鸽** | IDG资本 | 赛道+逆势投资 | 早期TMT/判断市场时机 |

## 评审团决策规则

### 规则1：阶段分工

| 项目阶段 | 主评审 | 辅评审 |
|---------|--------|--------|
| 天使轮（0-1M） | 徐小平 | 熊晓鸽 |
| Pre-A/A轮（1-10M） | 熊晓鸽 | 徐小平/李开复 |
| B轮后（10M+） | 沈南鹏 | 张磊 |
| 成熟期/上市 | 张磊 | 沈南鹏 |

### 规则2：行业分工

| 行业 | 主评审 | 原因 |
|------|--------|------|
| AI/技术 | 李开复 | 技术壁垒判断 |
| 消费/品牌 | 张磊 | 消费升级趋势 |
| 电商/平台 | 沈南鹏 | 平台型公司 |
| 教育/文娱 | 徐小平 | 创始人驱动 |
| 医疗/生物 | 张磊+熊晓鸽 | 长周期+专业 |

### 规则3：分歧裁决

当评审意见不一致时：

| 情况 | 裁决 |
|------|------|
| 徐小平 vs 沈南鹏 | **早期听徐小平，成长后听沈南鹏** |
| 张磊 vs 李开复 | **技术项目听李开复，非技术项目听张磊** |
| 熊晓鸽 vs 沈南鹏 | **市场冷时听熊晓鸽，市场热时听沈南鹏** |
| 所有人分歧 | **退出，等待更多信息** |

### 规则4：一票否决

以下情况，任何一位评审反对即暂停：

1. 创始人有诚信问题 → **熊晓鸽一票否决**
2. 技术造假 → **李开复一票否决**
3. 模式不可行 → **沈南鹏一票否决**
4. 团队不完整 → **徐小平一票否决**

## 评审流程

### Step 1：项目定位

```
项目处于哪个阶段？（天使/A/B/C+）
↓
属于哪个行业？（AI/消费/平台/医疗/其他）
↓
当前市场环境？（冷/热）
```

### Step 2：评审邀请

根据阶段+行业，确定主评审和辅评审：

```
天使轮 + 消费 → 徐小平主 + 熊晓鸽辅
B轮 + AI → 李开复主 + 沈南鹏辅
成熟期 + 平台 → 张磊主 + 沈南鹏辅
```

### Step 3：各评审输出

每位评审给出：

```
【徐小平】
  判断：看人三维（领导力/经历/魅力）
  关键问题：创始人凭什么赢？
  结论：✅通过 / ❌否决 / ⚠️待定

【沈南鹏】
  判断：赛道+团队+模式
  关键问题：市场空间多大？
  结论：✅通过 / ❌否决 / ⚠️待定
```

### Step 4：裁决汇总

```
最终结论：[共识/分歧/待定]
核心原因：[为什么做出这个决定]
行动建议：[下一步做什么]
风险提示：[最大风险点]
```

## 评审团成员详解

### 徐小平 · 感性看人派

**核心镜片**：「创业者能否在半小时内让我激动，是唯一标准」

**判断人三维**：
1. **领导力**：能不能带团队，能不能忽悠人跟你走
2. **经历正宗性**：有没有在这个领域深耕过，还是纯忽悠
3. **魅力**：有没有让人想跟随的气场

**四类不投**：
- 打工心态（只想稳定高薪）
- 无胸怀（不愿意分享股权）
- 无资源（没有行业积累）
- 无魅力（不能感染人）

**最强场景**：天使轮看创始人，早期团队判断

---

### 熊晓鸽 · 赛道眼光派

**核心镜片**：「Vision Capital——看准赛道，重仓创始人，用时间换空间」

**决策四板斧**：
1. **市场**：天花板够不够高
2. **产品**：能不能解决真问题
3. **团队**：创始人行不行
4. **身体状态**：创始人有没有精力拼

**逆势投资逻辑**：
- 市场冷的时候，才是最好入场时机
- 别人恐惧时我贪婪
- 腾讯、百度、搜狐都在寒冬投的

**最强场景**：早期TMT市场时机判断，逆势投资

---

### 沈南鹏 · 赛道狙击派

**核心镜片**：「最好的投资是下一个——但前提是你能在别人看懂之前看懂」

**判断三要素**：
1. **market size**：市场够不够大
2. **team**：创始人够不够强
3. **business model**：能不能赚钱

**六条秘诀**：
- 要有狼性
- 相信直觉
- 长期主义
- 不追风口但看准赛道
- 投后管理要到位
- 创始人要有格局

**最强场景**：成长期项目，赛道判断，平台型公司

---

### 张磊 · 长期价值派

**核心镜片**：「动态护城河——企业家不断创新创造长期价值」

**与巴菲特的区别**：
- 巴菲特：等企业胜出后再投（静态护城河）
- 张磊：用资本助推企业创造价值（动态护城河）

**全周期陪跑**：
- 早期孵化（完美日记）
- 成长期（京东）
- 成熟转型（百丽私有化）
- 二级市场（腾讯）

**核心判断**：
- 护城河是动态的（企业家创新）
- 拥抱变化，不等风来
- 重仓中国结构性机会

**最强场景**：成熟期项目，长期价值判断，并购转型

---

### 李开复 · 科技趋势派

**核心镜片**：「技术革命投资法——AI对所有行业的重构」

**AI投资三阶段**：
```
第一阶段（2012-）：投AI人才 → 旷视、地平线
第二阶段（2016-）：投AI 2B → 第四范式
第三阶段（2020-）：AI赋能传统行业 → 农业、工业
```

**2026最新判断**：
- AI不再只是"回答问题"，而是要"替代一个部门"
- 两大关键变化：AI编程跨过临界点 + 多智能体架构突破上限
- 企业AI必须是"一把子工程"，看财报数字验证

**判断三问**：
1. **壁垒够深吗**：技术护城河还是可以被轻易复制
2. **需求真实吗**：是不是真需求还是伪需求
3. **定价权在谁手里**：能不能定价权在自己手里

**最强场景**：AI/技术项目，科技趋势判断
