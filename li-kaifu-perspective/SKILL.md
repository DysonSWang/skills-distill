# 李开复投资视角（Li Kaifu Investment Lens）

> **核心镜片**：技术革命投资法——投的是"AI对所有行业的重构"，不是风口投机。

---

## 一、投资哲学

### 1.1 底层信念

**"AI 2.0是比移动互联网大10倍的机会"**

李开复认为，每一次重大技术革命都会重新定义人类生活和工作方式：
- PC时代 → Windows/Office
- 移动互联网 → 微信/抖音/美团
- AI 2.0时代 → 超级应用（还未出现）

**核心逻辑**：
- 不是投AI应用，而是投"AI+行业"的结构性机会
- 机会天花板极高，但平台公司风险极大
- 应用层风险更可控，回报更直观

### 1.2 投资阶段偏好

**早期投资（种子轮/A轮）为核心**，但会根据数据动态调整：
- 早期看赛道、看人
- B轮以后看核心指标（日活、付费用户）

### 1.3 核心方法论

**AI投资三阶段演进**：
```
第一阶段（2012-）：投资AI人才
→ 案例：旷视科技、地平线
→ 逻辑：人才稀缺，科学家稀缺

第二阶段（2016-）：投资AI 2B行业应用
→ 案例：第四范式、创新奇智
→ 逻辑：行业落地，赚钱效应

第三阶段（2020-）：AI赋能传统行业
→ 案例：农业AI、工业AI、医疗AI
→ 逻辑：+AI而不是AI替代
```

**三大投资方向**：
| 方向 | 特征 | 风险 | 回报 |
|------|------|------|------|
| 平台型 | 赢家通吃 | 高（千亿参数） | 极高 |
| 应用型 | 落地清晰 | 中 | 高 |
| 算力基础设施 | 刚需 | 低 | 稳定 |

## 二、决策框架

### 2.1 判断项目（项目维度）

**三问评估法**：

1. **壁垒够深吗？**
   - 做这件事比竞争对手有什么优势？
   - 技术壁垒 vs 商业模式壁垒

2. **需求真实吗？**
   - 是更好的替代品，还是创造新市场？
   - 避免"伪需求"

3. **定价权在谁手里？**
   - 上下游谈判能力
   - 价值链占比太低的没有护城河

### 2.2 判断人（创始人维度）

**最受VC追捧的创业者画像**：

```
1. 行业理解深度
   - 真正懂行业，不只是懂技术
   - 能把科研产品化

2. 强执行力
   - 产品能落地、能交付
   - 不只是"PPT创业"

3. 韧性+学习能力
   - 能扛过低谷
   - 能快速迭代

4. 致命问题排除
   - 不能把科研当baby（舍不得商业化）
   - 不能孤注一掷
   - 不能只追求估值
```

### 2.3 判断赛道（趋势维度）

**赛道选择标准**：
- 周期要长（不是短期风口）
- 天花板要够高（不是红海竞争）
- 能否创造新天地，而非抢固定蛋糕

**创新工场五大看好的领域**：
```
AI / 人工智能
文化娱乐
在线教育
B2B企业服务
消费升级
```

## 三、代表案例

### 3.1 经典成功案例

**字节跳动**
- 投资阶段：早期
- 投资逻辑：AI推荐算法 + 移动端先发优势
- 关键判断：张一鸣对算法驱动产品的深度理解

**美团**
- 投资阶段：A轮
- 投资逻辑：王兴对"本地生活"的长远判断
- 关键点：执行力极强的团队

**小米**
- 投资阶段：天使轮
- 投资逻辑：雷军的"铁人三项"模式
- 关键点：生态链打法，硬件+软件+互联网

**旷视科技**
- 投资逻辑：视觉识别人才稀缺
- 关键判断：科学家团队，技术领先

**第四范式**
- 投资逻辑：AI 2B落地
- 关键判断：行业Know-how比算法更重要

**知乎**
- 投资逻辑：高质量内容社区
- 关键判断：问答形式的平台效应

### 3.2 失败案例（教训）

**零售AI投资（安徽中商）**
- 教训：AI赋能传统行业比想象中难
- 问题：传统零售利润稀薄，AI改造投入大
- 警示：不要高估AI对低毛利行业的改造速度

**F5未来超市**
- 教训：无人零售概念失败
- 问题：技术成熟度不够，成本太高

## 四、AI判断与趋势

### 4.1 AI发展趋势（2024-2026）

**AI发展的四波浪潮**（出自《AI·未来》）：

```
第一波：互联网AI
→ 推荐引擎算法
→ 字节跳动、今日头条

第二波：商业AI
→ 商业智能化
→ 第四范式

第三波：感知AI
→ 语音识别、视觉识别
→ 旷视科技

第四波：自主AI
→ 自动驾驶、机器人
→ 还在路上
```

### 4.2 2026年AI判断（最新）

**核心观点：AI不再只是"回答问题"，而是要"替代一个部门"**

**2024年**：AI能否完成一个任务
**2025年**：AI能否完成一整条工作流
**2026年**：AI能否替代一个企业的职能部门

**两大关键变化**：
1. AI编程能力跨过临界点
   - 一年前：只能辅助写函数
   - 现在：端到端交付功能

2. 单一智能体有上限
   - 多智能体架构 = 突破天花板
   - 就像"企业委员会"，分工协作

### 4.3 企业AI转型判断

**核心金句**：
> "如果你的AI部署，最终没有改变任何一个出现在季度财报电话会议上的数字，那么你所做的就不是真正意义的AI转型，只是浪费钱打造了一个AI实验室。"

**真正值得AI介入的环节**：
- 收入
- 利润
- 供应链
- 动态定价

**转型失败的原因**：
- CIO自下而上推动 → 大概率失败
- 必须是"一把手工程"

## 五、金句摘录

### 5.1 投资哲学

**"不要把科研结果当成自己的baby"**
- 科学家创业的常见陷阱
- 要舍得商业化

**"当一个巨大浪潮来临的时候，如果能把握住最好的公司，就可能带来巨额回报，但是投资人必须有理性客观的判断"**
- 不要盲目追风口

**"不能把实验室的科研成果当成自己的baby，舍不得商业化"**
- 技术创业者的致命问题

### 5.2 AI趋势

**"AI 2.0时代，有三种不同的投资方法：平台型、应用型、算力基础设施"**
- 平台风险最高，应用最稳妥

**"整个AI大模型带来的机遇和技术浪潮，比过去Windows和安卓大10倍"**
- 天花板极高

**"AI不会是一个超级大脑的独角戏，而是更像不同智能系统协同运作的交响乐"**
- 多智能体是方向

### 5.3 创业者判断

**"创业者千万不能有的问题"**：
- 把科研当baby
- 孤注一掷
- 只追求估值

**"看创业者要看韧性和学习能力"**
- 能扛过低谷，能快速迭代

### 5.4 企业AI转型

**"传统CIO主导的自下而上AI转型模式大概率会失败，企业AI转型必须是'一把手工程'"**

**"2026年的竞争焦点不再是'AI能不能做到'，而是'谁能让AI真正替代一个部门、交付商业结果，并为这个结果负责'"**

## 七、禁忌事项

**不要做**：
- 投"表演式AI"（没有商业落地）
- 投只是追风口的AI应用
- 高估AI对低毛利传统行业的改造速度
- 投只懂技术不懂行业的科学家

**要做**：
- 看团队的行业经验
- 看产品能否落地
- 看财务数据验证（哪怕是早期指标）
- 看创始人是否舍得商业化
