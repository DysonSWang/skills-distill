## 吸引思维（核心起点）

**核心观点**：女人不是追来的，是吸引来的。

| 错误做法 | 正确做法 |
|---------|---------|
| 送礼物/请吃饭 | 展示自身价值 |
| 嘘寒问暖/关心讨好 | 让她欣赏你、崇拜你 |

**感动 ≠ 爱情**：

- 爱情：因为你有价值，她欣赏你、对你心动

**吸引的本质**：
- 你的事业心、主见、成熟恋爱观、稳重
- 她跟你交往起来会欣赏你、崇拜你、心动

## 7种思维模型

### 思维一：吸引思维

女人是被吸引的，不是追来的。感动不是爱情，爱情是相互吸引。

**反应模式（RAS系统）**：
- RAS = 网状激活系统，让一个人把注意力集中在有价值的事物上
- 你觉得女生漂亮有价值 → RAS要求你把注意力全集中在她身上
- 结果：你进入反应模式，对她的一切都反应激烈
- 反应模式 = 吸引力杀手

**反应模式的触发**：
- 跟朋友聊她 → 越聊越觉得她有价值
- 看她朋友圈 → 每条都研究
- 她发消息 → 秒回
- 她不回 → 一直等

**反反应模式**：
- 把自己当奖品，让她来追你
- 她追你 → 你是那个有价值的
- 你追她 → 她是那个有价值的

### 思维二：奖品思维

把自己当成奖品，让女生来追你。你是奖品，她才会想要赢得你。

**核心**：不是你去追她，而是让她来追你。

### 思维三：步骤思维

恋爱有固定流程，每步都不能跳：
```
吸引 → 舒适感 → 升高关系
```

### 思维四：前提思维

做事前要先有正确前提。前提错误，结果必然错误。

**追求的八个错误前提**：

### 思维五：真命思维

不要把女生当"真命"。越当真命，越容易失去自我。

**真命陷阱**：
- 过度关注她的一举一动
- 她的反应决定你的情绪
- 失去自我的跪舔

### 思维六：女神思维

女生也是人，不要把她神化。她也要吃饭、睡觉、上厕所。

**神化后果**：
- 说话变得拘谨
- 过度在意自己的表现
- 无法展现真实自我

### 博弈思维

**斌哥版博弈**：不是套路，是攻击性。核心是你要赢，不是要公平。

两性关系是博弈，你要赢，她也要赢。但你必须有攻击性，不能太卑微。

**关键区分**：博弈不是零和游戏，不是让她输。是你赢的同时，她也愿意跟你在一起。但你必须是主导的那个。

## 恋爱流程三阶段

### 第一阶段：吸引

**核心任务**：让她因为你本身的价值而喜欢你

**吸引来源**：
- 事业心、主见、成熟度
- 情绪刺激（让她觉得你有趣）
- 朋友圈、展示面
- 不暴露需求感

**斌哥流程案例**：
```
10:00 社交软件认识 → 看朋友圈觉得不错，产生初步吸引
10:00-11:00 聊天 → 展现情绪吸引，幽默风趣，不暴露需求感
11:00-12:00 继续增加吸引 + 建立舒适感
12:00 她答应下午3点邀约
```

### 第二阶段：舒适感

**核心任务**：让她了解你，建立信任和安全感

**必要信息交换**：
- 工作、职业
- 爱好
- 老家
- 童年趣事
- 人生观、价值观、婚姻观

**注意**：没有吸引做前提，单纯建立舒适感 = 老好人 = 备胎

### 第三阶段：升高关系

**核心任务**：肢体接触升级，从牵手到接吻到私密空间

**升级流程**：
```
牵手 → 接吻 → 私密空间
```

**失败处理**：
- 牵手失败 → 继续建立吸引 + 舒适感 → 再试
- 接吻失败 → 退回上一个阶段 → 再来

**斌哥案例**：
```
密室逃脱时拉起她的手（判断吸引+舒适感是否到位）
她没反抗 → 继续
逛完街后，在人少的公园问她...
```

## 心态建设

### 攻击性心态

**核心**：男人要有攻击性，不能太卑微

**没资格感**：
- 很多兄弟觉得自己没资格跟某个女生谈恋爱
- 原因：她的长相/身材/穿着/家境/朋友圈比你高一个层次
- 实质：怕暴露自己的真实状态

**攻击性表现**：
- 敢于表达自己的观点
- 不怕被拒绝
- 主动推进关系
- 敢于调侃和挑战

### 丰富心态 vs 稀缺心态

| 心态 | 表现 | 结果 |
|------|------|------|
| **丰富心态** | 觉得机会很多，她只是其中之一 | 淡定、有框架、吸引 |
| **稀缺心态** | 觉得她是唯一的，错过就没了 | 焦虑、跪舔、被拿捏 |

**核心**：资源池越大，心态越丰富。

### 场景自信 vs 核心自信

| 类型 | 定义 | 效果 |
|------|------|------|
| **场景自信** | 只在特定场景下自信 | 不稳定，容易崩塌 |
| **核心自信** | 内心深处相信自己 | 稳定，持续输出 |

## 关键概念解析

### 剥夺认同感（核心武器）

**定义**：不给她认同，逼她来追你。

**原理**：
- 女人身边有无数男人给她认同（点赞、讨好、送礼物）
- 她已经习惯了被认同
- 你不认同她 → 她反而会想你 → 开始投资

**剥夺认同感的方法**：
- 不秒回
- 不讨好
- 不送礼物
- 设置门槛/标准
- 让她来证明自己配得上你

**斌哥原话**：
> 你不断的认同她，她就不会把你当回事。只有两种结果——要么你变成供养者，要么她压根不跟你在一起。

**实战操作**：
- 她发朋友圈 → 不点赞
- 她主动找你 → 延迟回复
- 她示好 → 不立刻接受
- 让她猜测你的态度

### 赋予资格（配套技术）

**定义**：让她知道她是凭什么吸引你的，给她一个留下来的理由。

**时机**：当她情绪上已经对你投入很多之后

**方法**：
- 告诉她你喜欢她什么
- 赞美她的具体优点
- 让她觉得配得上你

**区别**：
- 剥夺认同感 = 让她追你
- 赋予资格 = 让她留下来

### 供养者 vs 情人

| 类型 | 特征 | 结果 |
|------|------|------|
| **供养者** | 用金钱/付出追女人 | 钱没了人就走，可替代性高 |
| **情人** | 用魅力/情绪吸引女人 | 真正吸引，不可替代 |

### 一手经验 vs 二手经验

| 类型 | 定义 | 例子 |
|------|------|------|
| **一手经验** | 你亲身体验过的 | 你追过100个女生，你知道什么有效 |
| **二手经验** | 别人告诉你的 | "对她好她就会喜欢你"——但你试了没用 |

**核心**：社会灌输给你的很多观念是错的。你必须通过一手经验验证。

**二手经验的危害**：
- 情人节送花 → 她反而更不珍惜你
- 送礼物追人 → 变成供养者
- 跪舔 → 被发好人卡

## 长期关系进挪（Phase 3详细版）

### 四个阶段

| 阶段 | 核心任务 | 进挪程度 |
|------|---------|---------|
| **初始阶段** | 陌生→认识→朋友 | 不建议进挪 |
| **相知阶段** | 普通朋友→亲密朋友 | 可以牵手拥抱，但不是必须 |
| **升级阶段** | 亲密朋友→暧昧 | 必须进挪，否则她觉得你没兴趣 |
| **确立阶段** | 确认关系 | 不要表白，让关系自然确定 |

### 各阶段操作

**初始阶段**：
- 在线聊天互动
- 为邀约做铺垫
- 不建议进挪，会显得猥琐

**相知阶段**：
- 达到目标：约会中友好互动、她主动分享生活工作
- 可以进挪，但不是必须
- 感知不到时机 → 不建议小白操作
- 牵手：直接牵，不要找借口，找借口等于暗示"我们的关系还没到"
- 牵了不要轻易甩开，除非她明显拒绝

**升级阶段**（关键！）：
- 必须进挪，不进挪她会觉得你对她没兴趣
- 可以亲亲抱抱
- 不要表白
- 可以说"我喜欢牵你的手"但让她自己猜
- 让她着急，对关系有好处

**确立阶段**：
- 不建议主动表白
- 等她百分之百确定喜欢你之后再表白
- 表白是确认关系，不是发起追求

## 长期关系危机处理

### 四种模式

| 模式 | 特征 | 结果 |
|------|------|------|
| **理智协商型** | 双方都理智，摊开来谈 | 最理想，但很少见 |
| **一方妥协型** | 一方一直妥协 | 常见，老好人模式 |
| **搁置争议型** | 先放着以后再说 | 隐患会累积 |
| **直接崩盘型** | 谁都不让步 | 最差结局 |

### 怎么往理智协商型发展

**核心原则**：第一次出现问题时就解决，不能拖。

**第一次危机处理**：
- 一定要摊开来说
- 即使有分手风险也要就地解决
- 拖下去 → 搁置争议型 或 一方妥协型

**表达顺序**：
1. 先陈述自己的错误
2. 再分析这件事
3. 最后指出对方的错误

**从一方妥协型转到理智协商型**：
- 一点一点让对方让步
- 下次她闹矛盾 → 拖一天再哄
- 条件性妥协：她给正反馈了，你再哄

**斌哥忠告**：
- 女人爱翻旧账
- 本来是小矛盾 → 可能勾起之前所有事
- 第一次就处理好，后面才顺利
