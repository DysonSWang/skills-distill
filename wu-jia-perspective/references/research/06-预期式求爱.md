## 一、什么是"预期"？预期如何管理对方心智？

### 1.1 预期的定义

**预期是一种让别人对你说的话感兴趣、愿意认真听你说话的技术。**

预期不是空穴来风，它是在**他人的心智层面**进行作战。搞社交、搞心理学、搞营销，都是在对方心智层面作战，不是我们自己心里想什么就做什么。

### 1.2 两种预期类型

预期分为两种：

| 类型 | 定义 | 使用场景 | 难度 |
|------|------|----------|------|
| **预期措辞** | 让别人认真听你说话的技巧 | 任何时候、任何地点、任何对话 | 简单，一学就会 |
| **标的预期** | 为了拉预期而去拉预期，针对特定目标 | 需要建立信任之后才能使用 | 较难，需要1年左右内化 |

### 1.3 预期措辞的两种手法

1. **设置问题**：先抛出一个对方感兴趣的问题，让对方带着好奇听你下一段话
2. **效果展示**：展示一个惊人的效果，让对方对你要说的内容产生期待

```
例子：
- 设置问题："你们想知道一个女人在高认识一个男人的时候，她最想知道的是什么吗？"
- 效果展示："大情圣六句话就可以让你跟女神劝了恋爱关系"
```

### 1.4 预期管理心智的核心

**预期管理心智的核心：通过设定问题或效果展示，调动对方的注意力，让对方对你要说的内容产生期待。**

关键点：
- 人不可能对没有需求的东西感兴趣
- 预期必须建立在对方的需求之上
- 效果展示是在拉对方的需求，跟对方的需求进行连接

## 二、预期的核心原理

### 2.1 第一性原理

预期理论的第一性原理是：

> **人在做出决定的时候，会有很多的恐惧感、不安全感、风险感。预期主要是为了打消对方的顾虑，解决对方的不安全感，最后能够跟对方成交或者跟对方合作。**

### 2.2 人脑的bug

预期措辞利用了人脑当中的一个bug：

> **人脑对于"增长信号"有天然的反应机制。**

当你展示一个可能让对方收益的结果时，对方的大脑会自动开始计算如何获得这个结果。

### 2.3 成交的核心原理

一切成交/合作都是先建立**人与人之间的信任**，再建立**人对事的信任**。

```
连招1.0 = 自我引导 → 有力描述 → 筛选 → 打压 → 休息 → 中段关系 → 亲密
核心 = 价值 + 信任 + 预期
```

**原理：**
- 你对于某个产品的信任，其实是建立在对人的信任上
- 明星代言的原理：我天天看成龙的电影，我相信成龙这个人，他推荐的东西我就会买
- 这就是"人对物的信任建立在人的基础上"

### 2.4 认知断裂理论

乌鸦提出了一个重要的概念：**认知断裂**。

```
需求 → 限制性条件 → 认知路径（解决方案）→ 新的需求 → 新的限制性条件 → ... → 认知断裂
```

**认知断裂点 = 商业价值的起点**

> 只有当人的认知发生断裂以后，才会开始寻求外部帮助，才有机会赚到他的钱。

## 三、预期的操作步骤/流程

拉动预期有**五个核心步骤**：

### 第一步：定位

**在做任何预期之前，先要找准对方的定位。**

核心逻辑：
- 没有人会被自己不感兴趣的事物拉预期
- 你必须先知道对方的行为倾向性
- 分析对方的**需求**和**限制性条件**
- 找到对方的**认知路径**（他解决问题的方式）

```
分析流程：
1. 展示高价值 → 观察对方对什么话题感兴趣（行为倾向性）
2. 如果聊女人他嗨了 → 他的认知路径在情感层面
3. 如果聊工作他嗨了 → 他的认知路径在商业层面
4. 分析他的限制性条件：他为什么还没解决问题？
```

### 第二步：忘记目的，建立信任

**在建立预期之前，先老老实实把人与人的信任建立起来。**

核心心法：
- 活在当下，珍惜眼前
- 先把价值感和信任感建立起来
- 不要脑子里装着两个目标（双手表效应）
- **做价值时不想信任，做信任时不想预期**

```
建立信任的流程（社交版连招）：
1. 展示价值（数字/身份/成就）
2. 建立舒适感（情感共鸣）
3. 交换信息/秘密

连招2.0是最接近社交版本的：
- 展示价值 → 直接建立舒适感
- 不需要筛选（筛选在社交场景会显得奇怪）
```

### 第三步：展示设计好的预期

**在信任建立之后，才可以把设计好的预期展示给对方。**

关键要点：
- 你不着急做预期
- 在做预期之前，你还在调整做预期的方案
- 对方已经清楚感受到那种吸引的能量
- 你自己没有任何紧迫感，铺垫过去

```
铺垫的技巧：
- 要有画面感：描述细节，加入大量修辞手法
- 不是为了过渡关系而说这个话，而是让这个说话看起来还像一个正常聊天
- 套路感消失的方法：加入大量的修辞手法之后，它就自动消失掉了
```

### 第四步：启动仪式

**在预期标的达成共识后，进行启动仪式，增加对方的违约成本。**

核心原理：
- 仪式是在心智层面直接催眠对方：**关系开始了**
- 仪式比写成文字更靠谱
- 最好有见证人（区块链效应：大家都知道这个事了）

```
人类历史上的各种仪式：
- 电影：开机仪式、杀青
- 师徒：拜师茶
- 中国古代：三寇酒拜、结拜兄弟
- 商业：签约仪式
- 恋爱：改口（叫老公/老婆）
```

**启动仪式的核心作用：增加对方的违约成本，让计划顺利实施。**

### 第五步：结束仪式（庆功宴）

**预期实现后，举行结束仪式，为这个预期打上成功的标签。**

核心价值：
- 让人们知道这件事完成了
- 成为你的一次成就，未来可以讲述的故事
- 让人们更加愿意相信你的下一个预期

```
结束仪式的例子：
- 庆功宴：请团队吃饭
- 公开课：反馈仪式，学员分享成功案例
- 课程结束：结业典礼
```

## 四、预期 vs 传统方法的核心区别

### 4.1 核心区别对照表

| 维度 | 传统方法 | 预期方法 |
|------|----------|----------|
| **销售/追求** | 直接挂产品/直接表白 | 先让对方产生想要的欲望 |
| **心态** | 推销（求着对方买） | 营销（让对方想要买） |
| **效果** | 阻力大、很累 | 阻力小、轻松 |
| **原理** | 销而不营 | **营而不销** |

### 4.2 "营而不销"的详解

```
传统销售：
产品挂上架 → 直接开卖 → 客户不想要 → 求着买 → 很累

预期营销：
前端营销工作做到位 → 用户已经非常想要 → 成交是自然结果 → 很轻松
```

**苹果的营销模式 = 浪潮式发售**
- 每次发售前做大量营销工作
- 不直接暴露销售意图
- 用户已经非常想买了再来

### 4.3 追求/表白也是一样

```
传统追求：
直接表白 → 暴露恋爱意图 → 对方产生顾虑 → 阻力增大

预期式追求（连招）：
自我引导 → 建立吸引 → 建立舒适感 → 对方想要这段关系 → 关系自然确立
```

**核心心法：**
> 不要让用户/女人意识到你想卖/想谈恋爱。当她意识到你想成交/谈恋爱的时候，阻力就会变得非常大。

### 4.4 需求感的本质

**需求感是一种毒药。**

当你满满的需求感的时候，你的价值是展示不出来的，你的价值展示行为一定会变成讨好行为。

```
为什么你展示不出价值？
因为你满脑子都在想怎么搞定对方
你满脑子都在想怎么占对方便宜
你的所有行为都会变味
```

## 五、预期在关系缔造中的作用

### 5.1 预期的核心作用

**预期是让对方看到"合作后的世界"，让对方开始幻想，上头，觉得太美了。**

预期让对方：
1. 对未来的合作产生美好的想象
2. 愿意为这个想象付出行动
3. 产生情感投入（投入越多，越难以离开）

### 5.2 画面感的三个层次

描述预期时要有画面感，画面感包含三个层次：

| 层次 | 说明 |
|------|------|
| 画面细节 | 具体场景的描述 |
| 个人感受 | 你在这个场景中的感受 |
| 哲学理解 | 对这件事本质的理解 |

**表达达到三个层次时，展现出来的东西一定是非常有画面感的。**

### 5.3 预期在连招中的位置

```
连招全流程：
自我引导 → 有力描述 → 筛选 → 打压 → 休息 → 中段关系 → 亲密

预期 = 中段关系 期间情理 的一种变形
预期 = 让对方提前看到关系的样子
```

### 5.4 关系设计的重要性

> **概念是被创造的，关系也是一种概念，所以关系也可以被创造。**
>
> **创造关系的方法跟创造概念的方法是一样的。**

乌鸦提出了"生存盟友"这个新概念，就是关系可以被创造的最佳例证。

**生存盟友的核心主张：**
- 既要亲密关系的支持
- 又不要亲密关系的限制
- 简单有效的强关系

## 七、预期式求爱实战流程图

```
强切 → 否定传统关系 → 模糊提出关系主张 → 展示高价值 → 情感共鸣 → 线下见面 → 风险承诺 → 回到线上 → 分享朋友案例（预期）→ 讲解概念 → 零风险承诺 → 确立关系 → 启动仪式
```

**每一步都是可以灵活调整的，关键在于理解原理而不是死守流程。**

---

## 八、核心心法总结

1. **预期必须建立在需求之上**，否则拉不动
2. **先建立人与人的信任，再建立人对事的信任**
3. **不要暴露需求感**，需求感是毒药
4. **仪式比文字更有效**，仪式催眠对方"开始了"
5. **结束仪式和启动仪式同样重要**
6. **降低自己的预期，升高对方的预期**
7. **预期是营而不销**，让对方想要，而不是你在推销
