# 乌鸦救赎 · 核心自信体系

## 核心自信定义

**核心自信** = 一个人很自在的状态，无所求，只是享受当下的状态。

- 佛教表述：活在当下，不被任何东西所牵绊，不被任何东西所影响
- 但不是变成木桶
- 能自然适应内心真实感受，并且还能说出恰当的话
- 核心自信最后的效果：万事万物不影响你的情绪，保持自在

**自在 > 自信**
- 自信会「心潮澎湃」，像听了摇滚音乐那种上头的感觉
- 自在是内心当中没有太大情绪波动，但感觉很舒服
- 目标：不管面对任何场景，都能让自己表现得非常自在、舒适，像是在自己家的后院一样

## 核心概念

### 1. 意志力可视化
- 自信不是来自熟悉领域（情景自信），而是核心自信
- 通过「对视」可以看见意志力
- 回避对视是缺乏意志力的表现 = 认怂
- 第一次见面，谁第一个回避目光，谁就是这段关系中的被带领者
- 谁目光更坚定，谁就更可能是带领者

### 2. 测试 > 感觉
- 「永远都不要相信自己的感觉，要相信自己去测试」
- 凭感觉会越猜越糟糕
- 测试是一种美德
- 例子：暗度是一种测试，问对方是否喜欢你

### 3. 自我提升
- 自我提升被具象化之后，就是一码技术
- 展示面自我提升 = 拍摄技术 + P图技术
- 变帅 = 护肤 + 整容
- 身材好 = 健身
- 条件不满足就去满足条件，不会就去学，差钱就去挣钱

### 4. 男神模式
- 不是套路，是生活状态
- 核心：让自己在任何场景都像在自家后院一样自在舒适
- 穿衣服主要看款式
- 不对自己设限

### 5. 连招体系
- 1.0 → 2.0 → 3.0 进化
- 自我排除：让自己先退出竞争，降低对方防备
- 猎心：捕获对方的心
- 强切：强势切入关系

### 6. 后宫模式
- 双女友模式
- 条件苛刻：智商条件+身体条件
- 很累，成功后更累
- 一夫一妻是保护男人的
