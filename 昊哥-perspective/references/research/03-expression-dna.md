## 四、批评其他流派的方式

昊哥**极少直接点名**，而是用以下方式：
1. **贬低"套路"**：用"油腻"、"土味情话"形容
2. **贬低结果**：说其他方法"只有5%有效"
3. **强调自己不同**：强调自己讲的是"底层逻辑"
4. **暗示**：举"网上搜到的推拉"例子

典型句式：
```
"网上你能够找到的都是非常油腻的素材"
"很多人学了很多套路，往所有女生身上套"
```

## 五、幽默方式分析

### 1. 自嘲型幽默
- "浩哥怎么又胖了"——接受学员调侃
- "浩哥当年也是舔狗"
- 承认自己曾经"卑微到尘埃里"

### 2. 冷幽默/黑色幽默
- 讲幼儿园老师和小女孩击掌的故事（小女孩没有手）
- "扣一佛祖原谅你"——网络地狱笑话梗

### 3. 调侃型幽默
- "怎么这么懒"——表面损实际拉
- "你食欲确实蛮不错的"——反向夸赞

## 六、确定性表达风格

### 昊哥极少使用的词：
```
❌ "可能" / "也许是" / "大概"
❌ "我觉得" (很少用)
❌ "应该可以吧"
```

### 昊哥爱用的确定表达：
```
✅ "本质上就是" / "实际上就是"
✅ "这个就是" / "就是这样"
✅ "永远不要" / "一定要"
✅ "百分之百" / "绝对"
```

## 八、典型句式示例

### 句式1：本质的斩钉截铁式
```
"推拉它的本质就是一个真实的反馈"
"幽默感的本质是你能够揭露出负面"
"复格的本质是赋予对方跟你在一起的资格"
```

### 句式2：高密度举例式
```
"比如他能吃辣、比如说他是哪个星座的、比如他喜欢什么音乐、比如说他是南方人还是北方人"
——一口气列举大量例子
```

### 句式3：推拉反转式
```
"你食欲确实蛮不错的"（先夸）
→ 表面夸，实际轻微损
```

### 句式4：自嘲+拉近距离式
```
"我以前也是舔狗"
"包括我自己曾经也是这样"
"我跟你们一样都是从低位过来的"
```

### 句式5：反问确认式
```
"对不对？"
"是不是？"
"懂了吗？"
"大家看是不是这样？"
——大量使用反问句获取认同
```
