## 二、核心操作模式总结

### 1. 筛选流程（标准化）

```
第一步：筛选位置
  → "你在武汉吗？"
  → 不在同一个城市直接下一个

第二步：筛选目的
  → "你玩朔是来找对象/玩伴/大字？"
  → 不管她说什么都要筛选

第三步：筛选年龄/照片
  → "你多大？不介意可以互换照片"
  → 不换就下一个

第四步：评价
  → 阿尔法男人姿态，对她进行评估
  → "看着挺成熟的，不像幼稚小女生"
```

### 2. 收号话术

- "加个微信，我也不想在这里聊"
- "你先"——用她先换照片来建立框架
- "有空可以出来玩"——不暴露需求感的邀约

### 3. 数约话术

**经典话术1：模糊邀约**
```
"晚上一起看电影"（实际想睡）
"不喝酒，喝点果汁就行"
"只是待在一起看看电影"
```

**经典话术2：给台阶**
```
"不是为了做那些事，做那事就没意义了"
"我觉得我像工具被用了一样"
"见面不喜欢就当交个朋友"
```

**经典话术3：画饼**
```
"最好的两人可以相依为命"
"在最累的时候有一个温暖怀抱"
"先亲后爱"
```

**经典话术4：破废物测试**
```
"你看我像海王" → "因为你事多吗"
"你喜欢我吗" → "你能事少一点我肯定更喜欢"
"比如呢" → 忽略，转移话题
"你对我有欲望吗" → "看感觉"
```

### 4. 现场操作

- 先开房，让她先上去，自己在房间等
- 不在楼下等，减少她的选择空间
- 进房间后先做自己的事（看电视、调空调）
- 等她卸下防备再推进
- 用"洁癖"借口先洗澡

## 四、底层逻辑提炼

### 1. 猎物思维 vs 追求者思维

- **追求者思维**：女人是高高在上的，需要追、需要讨好
- **猎物思维**：女人是猎物，我是在狩猎
- 核心表现：不解释、不道歉、不付出、框架硬

### 2. 框架博弈

```
女人给废物测试 → 你进入她的框架 → 被动
女人给废物测试 → 你用框架打回去 → 主动

"你觉得我像海王" → "因为你事多吗"
"你喜欢我吗" → "你能事少一点我肯定更喜欢"
"比如呢" → 忽略，不解释
```

### 3. 硬-软节奏

```
第一阶段：硬框架
  → 筛选、评价、给标准
  → 目的是打服她

第二阶段：软舒适
  → 给安全感、画饼、走心
  → 目的是让她愿意出来

注意：不能前面硬后面软，要保持一致性
```

### 4. 话术工具化

- 话术是工具，不是目的
- 目的是跳过过程，直接睡
- 说话要穿明白装糊涂
- "比如呢"是陷阱，不要回答，直接转移话题

### 5. 筛选精准度

- 收号不是瓶颈，瓶颈是"只有一根鸡吧操不过来"
- 时间是最贵的成本
- 筛选精准才能提高效率
- 不换照片就下一个，不纠结

## 五、矛盾与立场变化

### 关于舒适感与速度的矛盾

**早期（9月）**：追求速度，不要铺垫，直接约酒店

**10月**："不能太快，还是要铺垫"
→ 意识到纯速度会翻车，需要给安全感

**11月**："先硬后软，把问题过了再给舒适感"
→ 形成完整节奏：硬框架→打服→给舒适感

### 关于人设的立场

**早期**：展示面很重要，要有高端照片

**后期（技术流）**：
- "我朋友圈就发三个东西，连本人照片都没有"
- "虚假人设不重要，你又不会去研究她"
- "只要话术和技术够硬，不需要展示面"

### 关于自然流 vs 技术流

**早期学自然流**：通过展示面吸引

**转型**："自然流很扯蛋，完全可以靠话术干涉"
→ "技术流才是最牛逼的"

### 关于付出

**早期**：可能还会请吃饭、喝酒

**后期**："老子从来不请女的吃饭"
- "老子连奶茶都不请，请一瓶龙布就算够意思了"
- "能不花钱就不花钱，能白嫖就白嫖"

### 关于关系定位

**态度**：只想睡，不想谈
- "这个女人不适合做女朋友，适合睡一次就滚"
- "数约就是跳过的所有过程"
- "不谈恋爱，直接睡"

### 关于"喜欢"

**当女生问"你喜欢我吗"**：
- 90%的天狗会说"喜欢"
- 正确回答：不回答，反问："你觉得呢"
- 或者："你能怎么怎么样我肯定更喜欢"

## 六、被追问时的应对方式

### 问题1："你看起来像海王"
**错误回答**：解释、否认
**正确操作**：不解释，反问
→ "因为你事多吗"（暗示她事多，我不吃这套）

### 问题2："你喜欢我吗"
**错误回答**：直接说喜欢（暴露虚有感）
**正确操作**：不回答，反问
→ "那你呢？你对我什么感觉？"

### 问题3："比如呢"
**错误回答**：举例说明（掉进陷阱）
**正确操作**：忽略，转移话题
→ "那要两人一起探索了，只有我一个人想有什么意思"

### 问题4："你有什么目的"
**错误回答**：否认目的
**正确操作**：装糊涂，转移话题
→ "哎呀，只是看看电影喝点东西聊聊天"（睁眼说瞎话）

### 问题5："我们还不熟"
**错误回答**：道歉、降低要求
**正确操作**：给台阶，话术合理化
→ "见面前都不熟，见了面就知道了"

## 七、拒绝回答的问题类型

1. **追问具体细节**："你到底想干什么"——装糊涂
2. **废物测试**："比如呢"——忽略
3. **质疑框架**："你凭什么这么硬"——不解释
4. **要求承诺**："你会对我负责吗"——给话术但不给承诺

**核心原则**：管她说什么，不重要，阻挡不了我的步伐
