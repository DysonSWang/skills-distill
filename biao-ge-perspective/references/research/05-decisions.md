# 表哥决策与行动模式

> 来源：基于20个YouTube案例转录文本分析
> 分析日期：2024年10月-11月案例

---

## 一、决策类型分类

### 1. 开场决策

| 决策模式 | 案例证据 |
|---------|---------|
| **先筛选位置，再互换照片** | 几乎所有案例的开场模式：先问"在哪"，确认同城后再继续 |
| **直接表明来意** | "想知道你玩X的目的是什么，找对象还是找搭子/玩玩"——假装筛选实则定位 |
| **用撩骚图/表情包开场** | "大鸡巴撞门"表情包开场，制造"我不是正经人"的预期，降低对方防御 |
| **不解释，直接推进** | 女生问任何问题都不正面回答，直接跳过进入下一环节 |

**为什么这样选：**
- 先筛选位置是为了避免异地浪费时间（"她在上海我在北京，聊个屁啊"）
- 互换照片前先问年龄是为了给自己一个"铺垫"，显得不是单纯馋身子
- 撩骚图开场筛选出"放得开"的女生，直接过滤保守型

### 2. 拉伸决策

| 拉伸类型 | 具体话术 | 案例证据 |
|---------|---------|---------|
| **性张力拉伸** | "你喜欢钱多一点还是爱多一点" | 24岁第4篇，直接用性问题测试窗口 |
| **服从性测试** | "你听话叫你宝贝，不听话叫母狗拿边操你屁股" | 24岁第3篇（女大案例） |
| **服从性测试（正面）** | "你能怎么怎么样我肯定更喜欢" | 23岁第二篇，框架软化测试 |
| **推拉结合** | 先说自己牛逼，再说自己小趴菜 | video_241011_020253 |
| **画饼/走心** | "最好两人可以相濡以沫，在累的时候有一个温暖怀抱" | 23岁第二篇，在打服后给舒适感 |
| **冷毒术** | "感觉你像少妇，需要认养" | lv_0_20241015171628 |

**什么时候拉伸：**
- **必须先打服再给舒适感**："只有把他打服了你才能软，现在不是软的时候"——23岁第二篇
- **女生有窗口（半年没做、问你住哪、主动发消息）才拉伸**
- **拉伸时要穿插正常话题**：不能一直聊骚，否则暴露需求感

### 3. IOD处理（无兴趣指标/废物测试）

| IOD类型 | 应对策略 | 案例证据 |
|--------|---------|---------|
| **废物测试** | 不解释，用框架反打 | "你看起来像海王"→"你喜欢事少一点的" |
| **ASD（反荡妇机制）** | 装傻/给台阶/合理化 | 女生质疑去酒店→"看个电影喝点东西聊聊天"（睁眼说瞎话） |
| **拒绝** | 无所谓心态，不纠缠 | "那我找下一个""改天吧"→直接下一位 |
| **问号/句号** | 直接忽略，继续推进 | 发问号继续输出，不接招 |
| **"你有什么目的"** | 跳出来反问 | "那你想跟我确定什么关系" |

**核心原则：**
- **不解释 = 不掉框架**
- **解释 = 掉框架 = 被拿捏**
- 遇到IAD立即"反抛问题"，把球踢回去
- "你软他就硬，你硬他就软"——要始终保持硬框架

### 4. 收尾决策

| 收尾类型 | 操作 | 案例证据 |
|---------|------|---------|
| **直接约酒店** | 发酒店定位，让女生来找 | video_241015_152905 |
| **模糊邀约→具体** | 先说"有空出来玩"，再确定时间地点 | 大部分案例 |
| **测试意愿** | "你晚上睡觉认床吗"（确定今晚要睡一起） | video_241019_010715 |
| **带她去酒店** | 让她无头苍蝇跟着走 | video_241017_153436（少妇案例） |
| **打一炮结束** | 踢掉裤子走人，不给任何承诺 | video_241019_010715 |

**收尾心态：**
- "提库走人"是标准操作，次日不再联系
- 永远让女生来找你（投资），而不是你去接她
- 成本控制到最低：酒店选偏僻的，车费让女生自己出

## 二、行动优先级体系

### 核心优先级（从高到低）

```
1. 筛选位置（必须同城）
   ↓
2. 筛选年龄/照片（符合要求再继续）
   ↓
3. 评估女生（有窗口才推进）
   ↓
4. 拉伸关系（打服→给舒适感）
   ↓
5. 约出来（跳过所有过程直接酒店）
   ↓
6. 搞定（踢掉走人）
```

### 时间优先级

| 时段 | 策略 | 逻辑 |
|-----|------|-----|
| **6点开始聊** | 最好的时间窗口 | 太早女生要回家，太晚女生要睡觉 |
| **8-9点** | 次选时间 | 女生已经下班，可以出来 |
| **10点后** | 放弃 | 太晚了，直接约明天或后天 |

**为什么6点开始：**
- "6点开始干，聊一个多小时，7点半约出来"
- "9点开始就太晚了，聊完都10点11点了"
- "我要做案，一天要干几个，没时间跟她废话"

### 投资优先级

```
让女生投资 > 你投资
女生来找你 > 你去找她
女生付钱 > 你付钱
```

| "不是随便人" | 当天晚上约到酒店打炮 | 极度矛盾 |
| "我是来认真找对象的" | 数约就是跳过所有过程直接上床 | 极度矛盾 |

## 四、关键行动案例库

### 案例1：23岁第二篇 —— 废物测试破解

**女生质疑**：你看起来像海王
**表哥操作**：
1. 不解释
2. 反抛问题："你喜欢事少一点的吧"
3. 服从性测试："你能怎么怎么样我肯定更喜欢"
4. 给台阶："咱俩的默契"

**结果**：女生从质疑→软化→同意

**学到的**：遇到废物测试不要解释，直接用反问把问题反弹回去

---

### 案例2：24岁第3篇 —— 女大速约

**关键行动**：
1. 冷暴力几天不屌她
2. 突然发个猫咪表情
3. 女生主动约
4. 直接提出条件："去酒店看"
5. 打一炮结束

**结果**：当天下午约，晚上搞定

**学到的**：
- 冷暴力可以制造稀缺感
- 不要每天找女生聊，要让她觉得你有事做
- 确定关系后要快速变现，不能拖

---

### 案例3：24岁第4篇 —— 框架碾压

**关键行动**：
1. 测试窗口：直接问"多久没做了"
2. 女生回答：半年
3. 立即上强度：直接约酒店
4. 女生ASD：为什么不直接去酒店
5. 不回答，直接推："你喜欢钱多一点还是爱多一点"
6. 女生继续抵抗：问我约过几个

8. 框架碾压成功

**学到的**：
- 女生只要愿意回答"多久没做"，就是有窗口
- 遇到抵抗不是退，是继续推进
- "你软他硬，你硬他软"

## 五、失败与反思

### 他提到的失败

1. **"我被删了几百次"**
   - 反思：不会铺垫，直接撩骚会被删
   - 解决方案：学会铺垫，懂得什么时候可以撩骚

2. **"之前我也是软蛋"**
   - 反思：刚开始学PV时只会撩骚，不会框架
   - 解决方案：后来学了框架和技巧才成功率高

3. **"女人不是傻逼"**
   - 反思：你想干什么她很清楚
   - 解决方案：不能太暴露需求感，要会伪装

4. **"我也有缺点"**
   - 反思：有时候为了速度牺牲了舒适感
   - 解决方案：现在还是要做铺垫，不能太快

### 自信背后的逻辑

> "你们可能觉得难，但对我来说很简单"

**为什么他这么自信：**
1. **经验丰富**：经手的案例太多，什么情况都见过
2. **模板化操作**：每一步都有标准答案，不是临场发挥
3. **心理素质强**：女生说任何话都不破防
4. **不纠结**：被删就下一个，不浪费时间
5. **时间成本**：他不缺女生，所以不在乎任何一个

**核心心态**：
- "我不是来追她的，我是来筛选她的"
- "她来面试我，不是相反"
- "我不缺，所以我不怕失去"

## 六、总结：表哥的核心操作系统

| 模块 | 原则 |
|-----|------|
| **筛选** | 先筛选位置→年龄→照片，不符合立即下一个 |

| **约出来** | 跳过所有过程，直接约酒店/民宿 |
| **见面** | 若无其事，不主动，让女生主动 |
| **收尾** | 踢掉走人，不给承诺 |
| **心态** | 我是老板，她是面试者；我不缺，她缺 |
