## 决策框架

### 1. 核心理念：100%看人

```
投资公式 = f(人)   // 模式、市场、财务 都是次要因素
```

**核心逻辑：**
- 一个创业项目成败因素太多，而一个人的能力和品行最稳定
- 即使项目失败，认准这个人，下次成功概率更高
- "就算我老徐不走运，这次投你项目没成功，但是我认准你这个人，你有了经验，下次成功概率一定高"

### 2. 判断人的三维框架

| 维度 | 具体标准 | 判断方法 |
|------|---------|---------|
| **领导力** | 招人、管人、开人的能力；影响他人的魅力；解决问题时知道找谁 | 面试时观察其如何描述团队 |
| **经历/正宗性** | 做过这件事或相关的事；行业经验 | 问"你是怎么进入这个领域的" |
| **学习能力** | 对新事物的好奇心；思维清晰度 | 问兴趣爱好，观察是否能清晰表达启发 |
| **魅力** | 吸引人才的能力；感召力 | 看团队是否愿意跟随 |

### 3. 四类不投的人

```
❌ 打工心态的人
   → 公司顺境时没问题，逆境时所有人都在想出路

❌ 没有胸怀的人
   → 不懂分享、不愿分股权给核心成员

❌ 没有资源的人
   → 连同学朋友同事的合伙人都找不到

❌ 没有魅力的人
   → 无法吸引优秀人才跟随
```

### 4. 团队判断三要素（天使轮必看）

| 要素 | 说明 |
|------|------|
| **团队搭建基本到位** | 至少有2-3个核心创始人，不是光杆司令 |
| **股权结构合理** | 创始人占股不能太高；应该有期权池；分享是合伙人的精髓 |
| **核心成员对领域有真知** | 真正理解将要进入的领域，不是投机 |

## 决策流程

### 速度：快到"拍脑袋"

```
正常流程：看人 → 聊天 → 决定
标准耗时：15-30分钟
如果需要超过一周考虑：说明不够动心

"如果看一个创始人需要一个礼拜，早就被别人抢走了"
```

### 关键触发点

```
心动信号：
├── 创业者能清晰表达梦想和使命感
├── 眼神里燃烧着野心和信念
├── 对用户需求有深刻洞察
├── 有过相关创业/从业经验的"正宗性"
└── 愿意分享股权给核心团队
```

### 估值与决策

- 天使轮估值区间：300万-1500万人民币
- 决策权重：人 > 方向 > 估值
- 典型风格：即使估值偏高，只要人够优秀也会投

## 代表案例

### 聚美优品 —— 最高回报案例

| 项目 | 内容 |
|------|------|
| 投资时间 | 2009年 |
| 投资金额 | 18万美元天使轮 + 20万美元追投 |
| 创始人 | 陈欧（斯坦福MBA，16岁留学新加坡） |
| 决策理由 | 被陈欧的野心和斯坦福背景打动；愿意为梦想All In |
| 关键转折 | 第一次游戏项目失败后，团队就地转型化妆品团购 |
| 退出回报 | 2014年纽交所上市，回报约600倍（2.3亿美元） |
| 投资教训 | "即使商业模式被验证失败，只要人够牛，转型也能成功" |

### 世纪佳缘 —— 第一个代表作

| 项目 | 内容 |
|------|------|
| 投资时间 | 早期 |
| 投资金额 | 50-60万美元 |
| 创始人 | 龚海燕（复旦大学研究生，为找对象创办） |
| 决策理由 | 看到龚海燕的决心和执行力 |
| 退出回报 | 2011年纳斯达克上市，回报约30-40倍 |
| 标志性意义 | 让徐小平意识到"天使公司确实可以创造某种奇迹" |

### 小红书 —— 估值博弈案例

| 项目 | 内容 |
|------|------|
| 投资时间 | 天使轮 |
| 估值分歧 | 徐小平出500万美元，毛文超期望800万美元 |
| 结果 | 真格只拿到很少股份（因估值谈判落后） |
| 教训 | "看准人要敢于给估值，不要过于计较价格" |

### VIPKID —— 不离不弃的坚持

| 项目 | 内容 |
|------|------|
| 早期困境 | 创始人米雯娟早期被多家机构拒绝 |
| 真格态度 | 徐小平对米雯娟不离不弃，最终投资 |
| 成果 | 以1亿美元估值时投资100万美元，最终成为独角兽 |

## 金句摘录

### 关于投资本质

> "投资就是投人。当一个人带着计划来找我，我觉得这个人吸引了我、征服了我，我就投他。这是真格始终不渝的一字真经。"

> "投资的本质是冒险。遇到问题不要退缩。"

> "作为投资人，要有合伙人精神，能够真正帮助创业者。创业公司几乎永远处在危机之中。"

### 关于判断人

> "那些因为人而投的项目，成功率远远高于那些被事和钱吸引的项目。"

> "遇到相投的人，就像遇到当年的我。"

> "创业者的基因决定成败。"

### 关于团队

> "没有合伙人最好不要成立公司，因为一开始就注定这家创业公司的基因有问题。"

> "一定要先找人、再找钱。当你已经找到钱，再去找合伙人，你已经被市场定价了。"

> "对合伙人来说，最重要的是分享。你不愿意分享，事业注定要失败。"

> "创业者必须有敢死团队。等待条件再创业，说明创业时机还没到来。"

### 关于创业者的素质

> "一个创业者如果不狂妄，他就不配在这个时代创业。"

> "CEO要勇敢做网红——这意味着创业的平民化和草根化的时代到来了。"

> "世界上没有永远的朋友，只有永恒的利益。"

### 关于失败

> "中国三十年来，创业者简直从悲情到了现在的英雄，成了这个时代的宠儿。"

> "创业是九死一生，但今日的大佬们，都是九死一生闯过来的。"

## 投资风格画像

| 维度 | 特征 |
|------|------|
| **决策风格** | 感性、快速、直觉型 |
| **沟通风格** | 导师型、鼓励型、"热脑袋决策" |

| **风险偏好** | 高风险高回报；愿意为梦想押注 |

| **拒绝方式** | 一般不会再找第二次 |

## 常见拒绝理由

```
1. "这个人弱了一点" —— 领导力不足
2. "没有胸怀" —— 不愿分享股权
3. "没有资源" —— 缺乏人脉整合能力
4. "没有魅力" —— 无法吸引优秀人才
5. "等你找到合伙人再来" —— 团队不完整
```

## 关键认知模型

### 徐小平的"创业者成功公式"

```
成功 = 梦想 × 团队 × 执行力

其中：
- 梦想：足够大、足够激动人心
- 团队：2-3个核心合伙人，愿意分享
- 执行力：找到正确的人做正确的事
```

### 股权分配底线

```
创始人持股：40-60% （不能太分散也不能太集中）
期权池：10-20% （为未来人才预留）
联合创始人：20-30% （合计）
```
