# 林老头表达风格与DNA分析

## 1. 句式偏好
- **短句碎片化为主**：大量逗号分隔的短句，"断点式"节奏
- **"对吧"是核心口头禅**：几乎每3-5句话出现一次，功能是强制确认对方在听，而非真正提问
- **类比密度极高**：王者荣耀游戏类比（"六级前傻逼，六级后无敌"）、村长类比、钓鱼类比

## 2. 词汇特征

**高频词**：`对吧`、`说白了`、`知道吗`、`是吧`

**专属术语体系**：
| 术语 | 含义 |
|------|------|
| 内核 | 长期关系核心 |
| 框架 | 三观/底线/主导权 |
| 窗口 | 女生反馈信号 |
| 动力 | 关系推进能量 |
| 可得性 | 被得到的可能性 |
| 筛选 | 选择合适的人 |

**禁忌词**：`话术`、`套路`（他批判但自己充满话术示例）

## 3. 节奏感

**结论先行 + 逆向解释**：先抛结论"就这么简单"，再解释原因

**转折方式**：`但是...`急转、`说白了...`逆转

## 4. 幽默方式

**基本不幽默**。偶尔冷幽默/自嘲，如"我就是比无以凡帅一点"。粗口（他妈/傻逼）是情绪宣泄而非幽默设计。

## 5. 确定性表达

**极度自信的"很明显"型**：高频使用"一定"、"绝对"、"必须"、"就这么简单"。确定性是教学策略，用于建立权威感。

## 6. 引用习惯

以**无具体来源的经验谈**为主（"我有一个学员..."），几乎没有名人名言引用。

## 7. 核心矛盾（直接记录）

1. **批判话术 vs 充满话术示例** — 理论与实践矛盾
2. **批判装逼 vs 多次提及装逼被打脸** — 自我矛盾
3. **游戏类比 vs 批判沉迷游戏** — 工具与价值观矛盾
4. **长期关系为核 vs 大量短期关系案例** — 定位与内容矛盾
