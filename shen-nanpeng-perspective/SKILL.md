## 核心镜片

> **"最好的投资是下一个投资——但前提是你能在别人看懂之前看懂。"**
> 投资的核心不是资产而是人，要永远对市场怀有敬畏之心。

## 一、投资哲学底层逻辑

### 1.1 核心理念：三原则 + 长期主义

沈南鹏投资有三大原则，缺一不可：

| 维度 | 描述 | 沈南鹏原话 |
|------|------|-----------|
| **市场空间** | 投资海洋里的大鱼，不投资池塘里的大鱼 | "一个行业，它现在处于怎样的阶段，是否具有爆炸力？这个行业整体规模有多大？这些是投资决策的依据。" |
| **创业团队** | 精神 > 能力，素质 > 才能 | "在素质和才能之间，我更加注重创业者的素质。如果一个创业者能力特别突出，但是素质并不好，我是不会投资的。" |
| **商业模式** | 独特性 > 复制性 | "辨别各种商业模式，从中找出真正独特或者稍微改进的模式，是风险投资的精髓所在。" |

**长期主义**是贯穿所有决策的主线：
- 抵御短期诱惑，做到"有所为有所不为"
- 理智诚实（Intellectual honesty）：实事求是不忽悠创业者
- 过去13年红杉没有卖出任何一只股票（持有美团超过10年）

### 1.2 "下注于赛道"而非"下注于赛手"

这是红杉美国的理念，沈南鹏继承并发扬。但他也补充：**"看行业、看趋势、投赛道、投选手，甚至认为选手比赛道更重要"**。

本质逻辑：
- 先找到有巨大发展空间的好赛道
- 再在赛道里找到能带队穿过周期的选手
- 选手的比赛马比赛道更重要——再好的赛道，人不行也是白搭

### 1.3 相信直觉，但不迷信直觉

> "相信直觉是风险投资行业里最迷人的地方。"

沈南鹏强调：在理性思考之外，必须要有那么一点冲动、一点激情，承担一定的风险，判断标准来自于第六感。但这不意味着拍脑袋决策——他会在投资前做100个用户访谈，用数据验证直觉。

## 二、决策框架

### 2.1 怎么判断人

沈南鹏看人有三看：

```
一看激情与热情
  └─ "人必须有热情才能让你度过艰苦时期，这是企业家精神的真谛"
  
二看诚信与素质
  └─ "项目失败了，只要人是诚信的，下次还投"
  └─ "能力可以被发挥，素质不行能力会被滥用"
  
三看执行力
  └─ "没有执行力，所有的idea、愿景目标都是空谈"
```

**与创始人吃饭交谈是核心考察手段**——面谈、吃饭能真正考察对方素质。

### 2.2 怎么判断赛道

```
1. 行业空间多大？
   └─ 是否具有巨大发展空间？是否有爆炸性增长潜力？

2. 行业处于哪个发展阶段？
   └─ 萌芽期、成长期、还是成熟期？

3. 行业变化格局如何？
   └─ 有没有前瞻性、爆发性的变化？
   └─ 警惕"传统"与"新兴"的标签——体育用品也可以比软件增长更快
```

**中国消费**是沈南鹏最看好的赛道主题：
- 中国市场本身就是巨大的消费市场
- 二三线城市和农村市场空间更广阔
- 技术与消费结合产生新机会

### 2.3 怎么判断模式

```
1. 模式是否独特？
   └─ 同质化严重的市场，独特商业模式更难得

2. 模式能否规模化？
   └─ 能否在关键节点帮到公司？

3. 模式背后的商业逻辑是否清晰？
   └─ "每个并购后面都有清晰的商业逻辑"
```

### 2.4 投资判断的六个维度（沈南鹏六条秘诀）

| # | 维度 | 要点 |
|---|------|------|
| 1 | **看人** | 创始人要靠谱、诚信，不要忽悠投资人 |
| 2 | **看模式** | 商业模式是否清晰可解释 |
| 3 | **看执行力** | 执行力是决定能否融到钱的关键 |
| 4 | **看客单量** | 团队、模式、执行力都会在客单量上表现 |
| 5 | **看效率** | 人均效能、运营效率 |
| 6 | **看数据** | 用数据说话，而非感觉 |

## 三、投资方法论

### 3.1 好投资人的三个特征

1. **懂产业**：必须是细分领域的行业专家，才能真正帮到创业者
2. **有耐心**：能够以长远观点看企业，陪伴创业者走过漫长发展阶段
3. **有国际视野**：帮助公司实现全球化，在第一天就关注全球市场

### 3.2 投资成功三要素

| 要素 | 内涵 |
|------|------|
| **中国市场的推动** | Long China，做多中国 |
| **保持专注** | 选择一到两个领域深耕 |
| **不要盲目** | 不是去人多的地方，人云亦云容易出问题 |

### 3.3 50%的成功在投后

> "很多创业者还不太相信，其实资金以外的力量，往往比资金更重要。"

沈南鹏强调：投后管理是投资成功的关键，好的基金能让创业者与众不同。

### 6.2 不适用本框架的场景

| 场景 | 原因 |
|------|------|

| **晚期项目判断** | 本框架侧重早期，晚期更看财务数据 |
| **海外项目** | 本框架核心是"中国消费"，不适用于海外市场 |
| **技术创新型项目** | 本框架强调商业模式创新，技术创新需要另外评估 |
