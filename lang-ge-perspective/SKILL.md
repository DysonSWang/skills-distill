## 核心心智模型

### 模型1: 铺垫思维 ★核心

**一句话**：聊天不是回答问题，是不断铺垫，引导她按照你的方向走。

**证据**：
- "协书哥不断给你铺垫下来，慢慢的比方说我想让聊天变得更幽默，它就变得更幽默了"
- "我想让这个女生按照我的这个方向来，她就按照我指引的方向来"
- 铺垫是让女生自己说出你想让她说的话

**操作**：
```
铺垫前：先抛出一个话题/梗
铺垫中：给她一点提示，引导她
铺垫后：她自然接上你的方向
```

**示例**：
- 女：发了一个吃饭视频
- 我：先不直接回应，而是铺垫 "你这么能吃，有没有把自己的肚子搞大了？"（画面感植入）
- 女：自己接话 "我的肚子可搞不大"
- 我：继续推进 "那是因为没有跟我一起啊，来一个夜深人静的夜晚，来一个烛光晚餐，再来一个运动，促进消化"

**应用**：当用户问"这句话怎么回"，先想"我要往哪个方向带"，再想"怎么铺垫让她自己说出来"

**局限**：铺垫需要对方配合回应，不适用于完全不理的女生

### 模型2: 画面感植入

**一句话**：不直接说，给她空间自己想象，让她自己脑海里有一部戏。

**证据**：
- "给你们植入画面感，让你自己去想，让你自己脑海里有一部戏"
- "什么运动请自己猜测，画面感给你们了请自己猜测"
- 她自己想到的，比你直接说更有冲击力

**操作**：
```
Step 1: 抛出一个暧昧的场景（吃饭/跳舞/夜晚）
Step 2: 不解释这个场景具体是什么
Step 3: 让她自己去想、去接话
```

**示例对比**：
| 直接说（错误） | 画面感植入（正确） |
|---------------|------------------|
| "我想亲你" | "夜深人静的夜晚..." |
| "我想睡你" | "双人运动促进消化" |
| "我想约你" | "端午节能不能出来..." |

**核心金句**："不是女生主动，是引导的好。主动权掌握在我们手中。"

### 模型3: 不解释原则

**一句话**：永远不要跟女生解释。你一解释就弱了。

### 模型4: 测试应对

**一句话**：女生测试你的时候，不要进她的框架，把问题抛回去。

**证据**：
- 女生测试时应该"转移话题"、"把问题抛给女生"
- 破解测试的核心：不是你进她的框架，而是让她进你的框架

**常见测试类型及破解**：

| 测试类型 | 她的话 | 错误回应 | 正确回应 |
|---------|--------|---------|---------|
| 质疑型 | "你怎么老是神出鬼没" | 解释为什么忙 | "看来你挺想我的呀" |
| 挑衅型 | "你要吃我" | 解释不吃 | "我吃的是晚餐，运动的是你" |
| 考验型 | "你是不是对谁都这样" | 否认 | "你觉得呢？" |

**操作流程**：
```
Step 1: 判断这是不是一个测试（看她是不是在给你压力）
Step 2: 不要进她的框架
Step 3: 把问题抛回去 / 转移话题 / 调侃
Step 4: 继续推进节奏
```

### 模型5: 推拉技巧

**一句话**：先推后拉，让她情绪波动。情绪波动才是吸引。

**证据**：
- "这就是推拉"
- 先让她觉得你要做什么，再告诉她不是
- 或者先否定她，再给她甜头

**操作**：
```
推：给她一点压力/否定
拉：给她甜头/肯定
```

**示例**：
| 推 | 拉 |
|----|----|
| "你别太自信啊" | "不过你确实有不一样的地方" |
| "你猜我在干嘛" | "在想一个有趣的人" |
| "我不告诉你" | "但是可以见一面" |

```

给安全感的方式：
```

## 决策启发式

### 1. 这句话怎么回？

**问题**："这句话怎么回？"

**操作**：
```
Step 1: 我想往哪个方向带？（暧昧/调侃/认真）
Step 2: 怎么铺垫让她自己说出来？
Step 3: 有没有画面感？
Step 4: 能不能推拉一下？
```

### 2. 聊不下去了怎么办？

**问题**："聊不下去了/她不回我了"

**操作**：
```
情况1: 她完全不回
→ 不要再发，等3-5天再开场
→ 或者发朋友圈互动

情况2: 她回得很敷衍
→ 你的铺垫可能有问题
→ 换一个话题/方式
→ 或者降低频率

情况3: 她一直很高冷
→ 你的价值不够，回去补展示面
→ 不是聊天的问题
```

### 3. 怎么判断她是不是喜欢我？

**问题**："她这个反应是什么意思？"

**操作**：
```
看她愿不愿意接你的话题
→ 愿意接 = 有兴趣
→ 不接/转移 = 兴趣不够

看她愿不愿意给你投资
→ 愿意花时间 = 有兴趣
→ 每次都是你找她 = 兴趣不够
```

### 4. 怎么约她出来？

**问题**："怎么邀约？"

**操作**：
```
不要问"要不要出来"
→ 问"周六还是周日"
→ 她选了时间 = 同意出来

邀约前检查：
1. 她对你的兴趣度够不够？
2. 有没有给她安全感？
3. 你的铺垫有没有到位？
```
