## 能量守恒定律

**核心原则**：你投入的能量 = 她回复的能量

| 对方回复 | 你应该 |
|---------|--------|
| 多 | 你也多 |
| 少 | 你也少 |
| 快 | 你可以快 |
| 慢 | 你更慢 |

**能量守恒的目的**：
1. 不会暴露需求感
2. 保持自己的框架
3. 让她不会觉得你在追她

**能量守恒≠冷淡**：是保持对等的能量，不是故意冷落。

## 窗口识别（IoI/IoD）

**IoI = 兴趣指标**（女生对你说好听的话）：
- "你好帅"
- "你怎么这么有趣"
- 主动问你问题
- 主动约你

**IoD = 无兴趣指标**（女生对你说负面的话）：
- "滚"
- "再见"
- "你真烦"
- 不回复/已读不回

**中性话**（看不出好坏）：
- "哦"
- "好的"
- "嗯"

**窗口识别应用**：
- IoI → 推进关系
- IoD → 降低投入或撤退
- 中性话 → 保持现状，继续测试

## 聊天技巧

### 冷毒术

**定义**：用心理学方式描述她的性格，让她觉得"你很懂我"。

**冷毒的公式**：
```
"你看起来是XXX，但实际上你是XXX"
```

**示例**：
- "你看起来很高冷，但接触下来发现你其实很热情"
- "你看起来很独立，但其实内心很渴望被理解"

**注意事项**：
- 不要太准，太准像算命的
- 不要太假，假了像在套话
- 要自然，像朋友间的调侃

### 推拉技巧

**推**：先否定她，让她不舒服
**拉**：再给她甜头，让她舒服

**示例**：
- "你今天表现不错，但还差我一点点"（推+拉）
- "你挺有意思的，但就是有点二"（拉+推）

**目的**：制造情绪波动，让她对你上头。
