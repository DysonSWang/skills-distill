## 1. 核心公式

```
美女资源量 × 成义转化力 = 拿结果
```

Jason 强调要用**商业思维、理工经济学思维**对待把妹，如同操作一台精密机器。

---

## 2. 公式术语定义

| 术语 | 定义 |
|------|------|
| **美女资源量** | 前端收集的美女数量，即"子弹"，是实战的基础 |
| **成义转化力** | 后端转化能力，指收到微信号后的一系列转化操作 |
| **拿结果** | 最终的成功结果，包括短期关系、长期关系、发生亲密接触等 |

---

## 3. 前端与后端的分界线

**分界线：收到微信号**

```
前端（收到微信号之前）
├── 意识形态准备
├── 服化道
├── 展示面打造
├── 流量渠道（线上+线下）
└── 收号

后端（加到微信之后的一切）
├── 线上聊天 SOP
├── 线下约会 SOP
├── 转场技术
├── 私密空间操作
├── 破除 ASD
├── 得吃攻略
└── 长期/短期关系运营
```

## 4. 三大板块详解

### 板块一：意识形态准备（前置工作）

**核心：去婚化**

> "你不要跟我说什么，我还在脑子里还想着我要去结婚，我要花几十万财力娶到一个老婆，那我说你不要来学 game，你学 game 干嘛，你去相亲好了。"

- 去除传统婚姻思维枷锁
- 理解"谈恋爱和结婚是两回事"
- 建立正确的游戏心态

### 板块二：资源革命

**唯一目的：帮助加到尽可能多的美女资源**

> "巧妇难为无米之炊"、"资源是一切的基础"

### 板块三：后端转化

**解决：收到号之后怎么转化**

> "你收了一堆美女资源，收的妹子在微信里，你吃不到，转化不了，那就是太监逛青楼，没有用。"

## 5. 资源革命五大板块

| 序号 | 板块 | 具体内容 |
|------|------|----------|
| 1 | **服化道** | 个人形象、穿搭、发型 |
| 2 | **展示面打造** | 拍照、构图、场景选择、镜头语言、朋友圈发布 |
| 3 | **流量渠道** | 线上 + 线下 |
| 4 | **线上社交软件** | 积木、Tinder、牵手等 + 抖音小红书 IP 打造 |
| 5 | **线下街头搭讪** | 选址、话术、状态呈现 |

### 流量渠道细分

**线上：**
- 社交软件：积木、Tinder、牵手
- 抖音小红书：个人 IP 打造、流量获取、作品效能最大化

**线下：**
- 街头搭讪：地点选择、话术、状态呈现

## 8. Jason 背景介绍

| 项目 | 内容 |
|------|------|
| **身份** | 2025 顶级私教课程导师 |
| **背景** | 曾学习过大量情感课程、找过多个情感导师，深知碎片化学习的问题 |
| **教学特点** | 体系化、全流程、从前端到后端全套 SOP |
| **优势** | 不碎片化，强调知识融合，传授完整的框架体系 |

### 教学理念

Jason 认为现有课程的最大问题：
- 只教聊天
- 只放几个实战案例
- 碎片化、不系统

他的解决方案：
- 体系化教学
- 全流程覆盖
- 前端 + 后端完整 SOP
