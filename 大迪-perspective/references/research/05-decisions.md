# 大迪关键决策与行为模式研究

## 一、核心行为准则（8条）

### 1. 永远不要表白
- **准则**：「不表白关系才能持久，女生也没有拒绝你的机会」
- **逻辑**：表白是把主动权交给对方，一旦表白被拒，关系就陷入被动。不表白则永远保留退路
- **方法**：用约会和肢体接触代替表白，「就像打猎要么不动，一击必成」

### 2. 同时追多个，绝不只追一个
- **准则**：「千万不要只追一个，容易暴露需求感变得纠结，把对方越推越远」
- **逻辑**：只追一个会暴露强烈需求感，给对方压力；多线并进保持从容心态
- **方法**：广泛撒网，重点培养，心态不崩

### 3. 胆大心细脸皮厚，敢出手
- **准则**：「男人泡妞胆子一定要大，速战速决」
- **逻辑**：「饿死胆大的，撑死胆小的」，爱情本质是博弈
- **升级顺序**：「能轻轻就别抱抱，能抱抱就别拉手，能拉手就得搂搂腰」
- **关键**：约会三次后开始肢体接触，「逼她一把她才好做决定」

### 4. 聊天主动切断，制造下次机会
- **准则**：「聊天聊到最嗨的时候主动切断话题，下次她还会找你」
- **逻辑**：在高点结束留下悬念，让对方期待下次
- **禁忌**：聊天聊到没话题冷场，被对方掌握节奏

### 5. 永远不跪舔，行动大于言语
- **准则**：「甜狗甜狗一无所有」「低成本问候不如接她」
- **逻辑**：跪舔暴露需求感，越跪越不被珍惜
- **方法**：「作比说更重要」，用行动代替嘘寒问暖

### 6. 被拒一次立刻闪人，绝不死缠烂打
- **准则**：「只要被女生拒绝一次立马闪人，千万不要死缠烂打」
- **逻辑**：「你不喜欢她，她连屁都不是」，死缠烂打只会更被看不起
- **态度**：洒脱转身比持续追求更有魅力

### 7. 追女生不超过三个月
- **准则**：「追女生一定不要超过三个月，不成功赶紧白白」
- **逻辑**：窗口期有限，不行就换，效率优先
- **态度**：「不纠结不纠缠，下一个更乖」

### 8. 保持神秘感，保留30%
- **准则**：「永远保留30%的神秘，让女人捉摸不透」
- **逻辑**：「得不到的永远在骚动」，被拿捏就不懂得珍惜
- **方法**：有边界感，懂得保持距离

## 二、决策逻辑解析

### 底层逻辑：选择大于努力
- 「人以群分，选择大于努力」
- 方向比努力重要，选对赛道事半功倍

### 窗口期思维
- 女生喜欢你有窗口期，「会在这段时间特别特别喜欢」
- 窗口期不等人，过了这村没这店
- 不要以为她「慢热情」是在考验你，她只是对你兴趣不大

### 行动派哲学
- 「不要一直低头做所谓的准备，机会不会一直等你」
- 「任何闭门造车的准备都没有意义」
- 「你一直想成为心目中的完美男友，但比你穷的有一大堆，比你学历低的也很多」
- 准备永远没有完美的时候，先行动再说

### 得失心管理
- 「人无贪念必有馈赠，满心期待必有所失」
- 越在意越容易失去，保持淡然反而有吸引力
- 「心态越稳越淡然，吸引力你就越强」

## 三、做事顺序（进攻节奏）

### 标准流程
1. **约出来**：「大大方方的约出来，那就有机会，犹豫就会错过」
2. **三次约会后升级**：她愿意出来三次，说明有意愿
3. **肢体接触优先**：气氛好就牵手、亲，不要等
4. **用约会代替表白**：做完该做的，关系自然确立

### 约会升级路径
```
邀约 → 约会1-3次（观察反应）→ 牵手 → 搂腰 → 亲吻
```

### 聊天节奏
```
开场 → 曲解她的话制造有趣氛围 → 高潮时主动切断 → 等她来找你
```
