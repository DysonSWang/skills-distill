# 旭哥（情感导师）核心论点分析

## 一、反复出现的核心论点

### 1. 吸引优于追求
> "追妹子不是玩逻辑的，靠的是感觉，靠的是我身上散发这种吸引力"
> "靠吸引而不是靠追的"

### 2. 提供价值 vs 索取价值
> "你并没有给她提供价值，在她看来你是在向她索取价值的"

### 3. 感动 ≠ 喜欢
> "感动不是喜欢...感动产生的爱情并不牢靠"
> "你对他所做的一切，换来的只是他对你的感动"

### 4. 分手三类型及对应操作
| 类型 | 特征 | 操作原则 |
|------|------|---------|
| 失望型 | 对你有期望但未满足，有爱 | 少说多做，用行动证明改变 |
| 冷淡型 | 对你失去感觉，无爱 | 锻炼+展示自我，重新吸引 |
| 受伤型 | 被你某行为伤害，有爱 | 多说（代入感受）少做，扶平伤口 |

### 5. 得失心控制
> "你越想得到他，反而你越得不到他"

## 二、自创术语

| 术语 | 定义 |
|------|------|
| 硬价值 | 社会地位、收入、经济实力、生活品质 |
| 软价值 | 情商、智慧、上进心、经历等 |
| 行为模式 | 决定对方判定你高/低价值的依据 |
| 思维模式 | 与女生是否在同一频道 |
| 奖品 | 被追求者高高在上，追求者变成乞求者 |
| 得失心 | 对结果的过度执念，导致行为变形 |

## 三、思维框架体系

### 五维吸引模型
```
硬价值 → 软价值 → 行为模式 → 思维模式 → 心态
```

### 核心金句
- "寓豪而不是靠追的，而是靠吸引"
- "懂比爱更重要"
- "得到的不被珍惜，得不到的永远在骚动"
- "你越想得到他，反而你越得不到他"
