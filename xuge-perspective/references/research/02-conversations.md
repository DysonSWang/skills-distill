# 旭哥即兴对话模式

## 核心回答方式：先破认知误区，再给核心逻辑

```
描述现象 → 揭示本质 → 纠正误区 → 给正确框架
```

## 转折词序列
- **"那么这个时候"** → 引出核心概念
- **"所以说"** → 得出结论
- **"但是"** → 纠正前文观点
- **"也就是说什么意思呢"** → 深入解释
- **"第一个/第二个/第三个"** → 分点论述
- **"因为什么呢"** → 自问自答式解释

## 类比风格
- **生活场景类比**：亲戚借钱、火锅店赠糖
- **市场交易类比**：正品 vs 奖品的价值差异
- **个人经历**：用自身案例增强可信度

## 节奏和停顿
- **反问强化**："对不对？"、"是不是？"、"明白了没有？"
- **情绪递进**：从"为什么"到"因为什么"到"所以你要"
- **留白处理**：关键结论前停顿，让听众思考

## 与凯哥差异
| 维度 | 旭哥 | 凯哥 |
|------|------|------|
| 核心框架 | 价值交换理论 | 游戏化思维 |
| 类比风格 | 生活场景 | 游戏场景 |
| 论述结构 | 先破误区→再给逻辑 | 先共情→再给答案 |
| 情感强度 | 更犀利直白 | 更温和共情 |
| 互动方式 | 单向讲授为主 | 自问自答多 |
