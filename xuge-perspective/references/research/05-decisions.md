# 旭哥决策框架与行动指南

## 快速规则（如果X则Y）

| 条件 | 规则 |
|------|------|
| 如果你越对一个女人好 | 她反而越躲着你（因为你在索取） |
| 如果她不喜欢你 | 你的付出换来的只是感动，不是喜欢 |
| 如果她不回消息 | 不要追问，不要纠缠 |
| 如果你要约她 | 提前安排好，直接告诉她行程 |
| 如果聊天没话题了 | 不要硬聊，转场或改天再聊 |
| 如果你发现你总是追不到 | 把买礼物的钱给自己买衣服 |
| 如果你要表白 | 先铺垫情感，否则只会让她觉得你油腻 |
| 如果你想挽回 | 不要道歉，不凡是求和好，要重建舒适感 |

## 红线（绝对不做）

1. ❌ 不要表白后立刻讨好
2. ❌ 不要每天24小时找她聊天
3. ❌ 不要送礼物后立刻提出约会要求
4. ❌ 不要无条件付出
5. ❌ 不要把自己贬低
6. ❌ 不要问"你什么时候有空"
7. ❌ 挽回时不要道歉求和好
8. ❌ 不要跳环表白

## 时机判断

| 时机 | 判断标准 |
|------|---------|
| 表白时机 | 取决于她心里有没有你 |
| 聊天升温 | 聊到某个话题时，插入"我们两个+暧昧动作" |
| 约会惊喜 | 用"反差感"——先给冰水再给热奶茶 |
| 挽回时机 | 等她重新感到舒适时，不用你提和好 |
| 尴尬转场 | 一旦感觉尴尬，立刻换场地 |

## 分手类型对应操作

| 类型 | 操作 |
|------|------|
| 失望型 | 少说多做，行动证明改变 |
| 冷淡型 | 锻炼+展示自我，重新吸引 |
| 受伤型 | 多说（代入感受），少做 |
